常见问题

  1. 数据库连接失败

    • 确保 PostgreSQL 正在运行
    • 检查 .env 文件中的数据库 URL

  2. 前端构建失败

    • 确保安装了 trunk 和 wasm32-unknown-unknown target
    • 检查 Node.js 和 Sass 是否正确安装

  3. CORS 错误

    • 确保后端 CORS 配置正确
    • 检查前端 API 请求的 URL
