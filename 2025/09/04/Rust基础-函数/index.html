函数的组成（fn、函数名、参数、返回值）

函数的组成

  • 声明函数的关键字fn
  • 函数名 add
  • 参数 i 和 j 和参数类型 i32
  • 返回值类型 i32
  • 函数体 i + j

fn

  • 函数的位置可以随便放，Rust不关心我们在那里定义了函数，只要有定义即可

函数名

  • 开头是字符，后面可以是字符、数字、下划线
  • 开头是下划线，后面可以是字符、数字、下划线，不能仅有下划线
  • 函数名和变量名使用蛇形命名法，即snake_case，例如fn add_number() -> {}

函数参数

  • Rust是强类型语言，因此需要你为每一个函数参数都标识出它的具体类型
  • 参数类型和参数名之间用冒号:分隔

函数返回

  • 函数的返回值就是函数体最后一条表达式的返回值，不需要使用return提前返回
    • 想要完全理解这句话需要掌握**语句（statement）和表达式（expression）**的区别
    • 语句是执行操作的，没有返回值
    • 表达式有返回值
    • 语句以分号;结尾
    • 表达式不以分号;结尾
  • 初学者就记住函数的两种形态就行了：
    • return + 分号
    • 没有return + 没有分号
  • 函数返回值类型使用->进行标识
